## 🚀 실행 방법

1. **저장소 클론**
   ```bash
   git clone https://github.com/your-username/cusor_HomePage.git
   cd cusor_HomePage
   ```

2. **로컬 서버 실행**
   ```bash
   # Python 3
   python -m http.server 8000
   
   # 또는 Node.js
   npx serve .
   ```

3. **브라우저에서 확인**
   ```
   http://localhost:8000
   ```
